# 重说纠正

## 模式

说错后立即纠正，删除前面错误的部分。

### 1. 部分重复

前后词语有重叠但不完全相同：

| 原文 | 删除 |
|------|------|
| 你再关你关掉 | "你再关" |
| 怎么让它去有个更大的 | "怎么让它去" |

### 2. 否定纠正

用否定词纠正刚说的：

| 原文 | 删除 |
|------|------|
| 它是它不是 | "它是" |
| 可以不可以 | "可以" |

### 3. 词被打断

词说一半 + 静音 + 重说完整：

| 原文 | 删除 |
|------|------|
| 依赖[静]依赖关系 | "依赖[静]" |

## 检测逻辑

```javascript
// 找相邻词的公共前缀
if (word[i].text.startsWith(prefix) && word[i+n].text.startsWith(prefix)) {
  // 且后者更完整 → 删前者
}
```
