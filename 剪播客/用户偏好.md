## ⏱️ 时长偏好

### 目标时长
```yaml
理想时长: "90分钟"   # 较长内容，深度对话
最长可接受: "105分钟" # 理想时长+15%
最短可接受: "75分钟"  # 理想时长-15%
删减激进度: "保守"    # 只删除明显冗余（10-20%删减）
```

**说明**：
- **保守**：只删除明显冗余（10-20%删减）
- **中等**：平衡精简和完整性（20-35%删减）
- **激进**：追求极致精简（35-50%删减）

## 📝 使用说明

### 首次使用
1. Claude会询问完整的偏好设置
2. 填写"基本信息"和"播客定位"
3. 设置默认偏好
4. 后续使用会自动应用这些偏好

### 日常使用
1. 告诉Claude音频文件和说话人
2. Claude会读取你的偏好
3. 如果有特殊需求，可以临时调整
4. 选择是否保存本次调整为新默认

### 更新偏好
你可以随时编辑此文件，或者告诉Claude：
- "更新我的默认时长为90分钟"
- "我现在偏好激进删减"
- "不再自动删除口癖"
