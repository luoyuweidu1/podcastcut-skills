# 连续填充词

## 模式一：两个语气词连在一起

```
嗯啊、啊呃、哦嗯、呃啊
```

## 模式二：句首语气词串（≥3个填充词/确认词连续）

**定义**：句子开头由语气词+确认词（对、是、好）组成的连续串，无实质信息。

```
嗯嗯，呃，对，就是…
嗯，对对对，然后…
啊，呃，好，就是说…
```

### 检测逻辑

```javascript
const fillerWords = ['嗯', '啊', '哎', '诶', '呃', '额', '唉', '哦', '噢', '呀', '欸'];
const confirmWords = ['对', '是', '好', '行', '嗯嗯'];

// 模式一：两个语气词相邻
if (fillerWords.includes(curr) && fillerWords.includes(next)) {
  markAsError(curr, next);
}

// 模式二：句首连续填充+确认词串（≥3个）
const allFiller = [...fillerWords, ...confirmWords];
function checkSentenceStartFillerRun(sentenceWords) {
  let runLen = 0;
  for (const w of sentenceWords) {
    if (allFiller.includes(w.text)) {
      runLen++;
    } else break;
  }
  if (runLen >= 3) {
    // 删除前 runLen 个词
    markAsError(sentenceWords.slice(0, runLen));
  }
}
```

### 案例

| 原文 | 连续填充词 | 删除 |
|------|-----------|------|
| 嗯嗯，呃，对，就是我 burn out 的一次 | 嗯嗯+呃+对 (3个) | "嗯嗯，呃，对，" |
| 嗯，对对对，然后我去了 | 嗯+对+对+对 (4个) | "嗯，对对对，" |
| 啊，呃，好，就是说我们 | 啊+呃+好 (3个) | "啊，呃，好，" |

## 删除策略

全部删除。
